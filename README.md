Health Tracker | React, Node.js, Redux, MongoDB
- Built a health tracking application with secure user registration and personalized dashboards for detailed wellness monitoring.
- Developed features to log and edit daily health metrics like weight, caloric intake, total sleep, and steps.
- Implemented a data visualization page to display long-term health trends via clear and simple charts.
- Created a dedicated blog module for users to maintain a digital journal and document their daily progress of their health journey.
